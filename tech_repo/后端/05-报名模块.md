## 做了什么

实现了整个报名提交流程的核心逻辑，一个请求进来的完整链路：

```
前台用户点击「提交报名」
    ↓
POST /api/v1/registrations（NestJS 微服务）
    ↓
RateLimitGuard 检查（超频？→ 返回 429）
    ↓
ValidationPipe 校验 DTO（格式错？→ 返回 400）
    ↓
RegistrationController.create(dto)
    ↓
RegistrationService.create(dto)
    ├── 查询 Strapi：活动是否为 active？（否 → 400）
    ├── 查询 Strapi：是否已达容量上限？（是 → 400）
    ├── POST 到 Strapi 写入 Registrations 表
    └── 加入 email 队列（异步发确认邮件）
    ↓
返回 { success: true, id: "xxx" }（201）
```

## create-registration.dto.ts 详解

DTO（Data Transfer Object）定义「请求体应该长什么样」：

```typescript
export class CreateRegistrationDto {
  @IsString()
  @IsNotEmpty({ message: '活动 ID 不能为空' })
  eventId: string           // 报名哪个活动

  @IsObject()
  @IsNotEmpty({ message: '表单数据不能为空' })
  userInfo: Record<string, string | number>  // 动态表单填写的内容
}
```

**什么是 class-validator：**  
`@IsString()`、`@IsNotEmpty()` 这些是装饰器，告诉 NestJS 的 ValidationPipe 如何校验字段。  
如果请求体里 `eventId` 不是字符串，或者 `userInfo` 是空的，ValidationPipe 自动返回 400。

**合法的请求体长这样：**
```json
{
  "eventId": "abc123",
  "userInfo": {
    "name": "张三",
    "email": "zhangsan@utoronto.ca",
    "year": "大一"
  }
}
```

## registration.controller.ts 详解

Controller = 路由层，只负责「接请求、调 Service、返结果」：

```typescript
@Controller('registrations')
export class RegistrationController {
  @Post()
  @HttpCode(201)           // 成功返回 201 Created，不是默认 200
  @UseGuards(RateLimitGuard)  // 先过防刷守卫
  async create(@Body() dto: CreateRegistrationDto) {
    const result = await this.registrationService.create(dto)
    return { success: true, id: result.id }
  }
}
```

**`@UseGuards` 的执行顺序：**  
Guard 在 Controller 方法之前执行，Guard 返回 false 或抛出异常，请求就不会进入 create()。

## registration.service.ts 详解

Service = 业务逻辑层，是最核心的地方：

```typescript
async create(dto: CreateRegistrationDto) {
  // 第一步：查活动，验状态
  const event = await this.fetchEvent(dto.eventId)
  if (event.status !== 'active') {
    throw new BadRequestException('报名通道已关闭')
  }

  // 第二步：检查容量
  if (event.capacity !== null) {
    const count = await this.fetchRegistrationCount(dto.eventId)
    if (count >= event.capacity) {
      throw new BadRequestException('报名人数已满')
    }
  }

  // 第三步：写入 Strapi
  const registration = await this.createRegistration(dto)

  return { id: registration.id }
}
```
